About My Netflix Clone

This project is a Netflix-inspired web application built using Django and Bootstrap. It features a dynamic homepage with a movie carousel and responsive cards displaying trending films. Each movie card links directly to the official YouTube trailer for instant viewing.

The website also includes a fully functional Contact Page where users can submit their name, email, phone number, and message. All form data is validated before submission and securely stored in the Django Admin portal, where it can be viewed, updated, or deleted by the admin.
